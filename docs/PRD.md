# 玉龙VPN Windows 客户端 PRD

## 1. 产品定位

玉龙VPN Windows 客户端是玉龙VPN生态里的电脑端客户端，用于给 Windows 用户提供更简单的一键代理连接体验。

## 2. 目标用户

- 已经使用玉龙VPN Android 的用户。
- 需要电脑端连接的用户。
- 不懂 Clash 配置、不想手动导入配置的小白用户。

## 3. 核心需求

用户只需要做一件事：

```text
输入动态验证码 → 登录成功 → 一键连接
```

## 4. 第一版功能

| 编号 | 功能 | 说明 |
|---|---|---|
| 1 | 登录动态验证码 | 登录前只能看到输入密码页面 |
| 2 | 显示到期时间 | 登录成功后显示后台返回的 expiresAt |
| 3 | 显示后台公告 | 从 notices 接口读取公告 |
| 4 | 一键连接 | 下载配置、启动核心、设置系统代理 |
| 5 | 一键断开 | 停止核心、关闭系统代理 |
| 6 | 节点列表 | 从配置里提取节点名显示 |
| 7 | 自动选择节点 | 第一版显示自动选择，后续接策略组 |
| 8 | 系统代理开关 | 手动开启/关闭 Windows 系统代理 |
| 9 | 开机自启 | Tauri autostart 插件实现 |
| 10 | 托盘图标 | 支持显示、连接、断开和退出 |
| 11 | 自动更新配置 | 重新请求 clash-config |
| 12 | 软件状态 | 显示未登录、已登录、连接中、已连接、异常 |

## 5. 第一版不做什么

第一版暂不做：

```text
TUN 全局模式
真实虚拟网卡
付费系统
软件自动升级签名
复杂测速
多语言
```

这些放到 v2 / v3。

## 6. 后台接口

复用 Android 端：

```text
POST /api/access-code
GET  /api/clash-config?code=验证码
GET  /api/notices?public=1
```

## 7. 成功标准

v1 成功标准：

```text
1. GitHub Actions 能打包出 Windows 安装包
2. EXE 能打开
3. 能输入验证码登录
4. 能显示公告和到期时间
5. 能下载 clash-config
6. 能开启/关闭系统代理
7. 安装包内置官方 mihomo v1.19.28 并能启动代理核心
```
